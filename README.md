# nodejs-web-api-fastify-redis-set

## Description
A basic web api looksup and returns a set.

## Tech stack
- nodejs
  - fastify
  - redis

## Docker stack
- node:latest
- redis:latest

## To run
`sudo ./install.sh -u`
- http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
